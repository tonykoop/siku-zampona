# learn-to-play / Siku/Zampona (G)

Starter songbook for the G siku/zampona panpipe from the build packet. Generated by the sheet-music skill.

The siku splits notes across pipe rows and often across two players. These studies keep the hocket pattern visible and beginner-friendly.

Range: **G4 - B5 core range, with optional upper-row extensions.**

## Folder map
| Folder | Contents |
|---|---|
| `00-warmup-scales/` | Row-to-row range walk, thirds hocket, breath-reset drill |
| `01-easy/` | Three beginner melodies arranged for alternating pipe rows |
| `02-intermediate/` | Two hocket studies for paired ira/arca practice |
| `03-original/` | *Reed Rows Round* - Heifer Zephyr original |
| `fingering-charts.svg` | Pipe-row map and G pentatonic fingering chart |

## Notes on notation
ABC files use concert pitch. Notes tagged in comments as ira/arca should be split between paired players where possible.

## L2 readiness and known gaps
This starter songbook is ready for L2 review as a playable packet draft: ABC melodies, per-tune notes, static fingering summaries, warmups, and a top-level chart are present. Known gaps: generated audio, MusicXML, LilyPond, MIDI, and printable songsheets are skipped until the shared render dependencies are available; fingering charts need builder review against the final hole layout.
